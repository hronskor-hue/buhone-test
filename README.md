# BuhOne — Бухгалтерские услуги в Санкт-Петербурге

Тестовое задание на позицию верстальщика.  
Адаптивная вёрстка лендинга по макету Figma.

## 🚀 Демо

[Посмотреть живую страницу](https://hronskor-hue.github.io/buhone-test/)

## 🛠 Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, адаптив, анимации)
- JavaScript (ES6, без фреймворков)
- Методология БЭМ (именование классов)

## 📋 Функционал

- Полностью адаптивный дизайн (mobile, tablet, desktop)
- Рабочий слайдер в hero-секции (переключение по точкам)
- Плавная навигация по разделам страницы
- Форма обратной связи с валидацией полей
- Всплывающее окно (popup) об успешной отправке
- Валидный HTML/CSS (проверено W3C)

## 📂 Структура проекта

- `index.html` — главная страница
- `css/style.css` — стили
- `js/script.js` — логика слайдера, навигации и формы
- `images/` — изображения

## ⏱️ Затраченное время

~1 час 30 минут (вёрстка) + доработки по обратной связи.
