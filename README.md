## 📝 Тестовое задание на позицию верстальщика

### Исходное задание
Сверстать адаптивный макет по ссылке:  
https://www.figma.com/file/TO7JfO39gafxdkwwmmUSMY/Templates-%236.-More-on-Figma.info-(Copy)?node-id=2%3A2

**Требования:**
- Без использования фреймворков (Bootstrap и аналоги)
- Чистый валидный код (HTML/CSS/JS)
- Адаптивная вёрстка (mobile, tablet, desktop)
- Все поля формы обязательны
- Попап об успешной отправке формы
- Результат выложен на GitHub (репозиторий + GitHub Pages)

### 🚀 Демо

### 🛠 Технологии
- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, адаптив, анимации)
- JavaScript (ES6, без фреймворков)
- Методология БЭМ (именование классов)
